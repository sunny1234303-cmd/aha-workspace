# Commands

사용 가능한 Slash Commands 목록입니다. (7개)

## 초기 설정

- `/setup-workspace` - 워크스페이스 초기 설정 (CLAUDE.md + 데이터베이스 자동 생성)

## Daily Workflow

- `/daily-note` - 오늘 기록 생성/열기
- `/todo` - 할일 캡처/조회 (daily 기록에서 자동 추출도 겸함)

## AI 활용

- `/gather` - 정보 수집 모드
- `/reframe` - 이해 확인 모드
- `/truth` - 사실 기반 분석 모드

## 지식 관리

- `/idea [카테고리]` - 아이디어 저장

---

## Claude Code 기본 내장 커맨드

아래는 이 워크스페이스가 만든 게 아니라 Claude Code 자체에 있는 커맨드입니다. 자주 씁니다.

- `/compact` - 대화가 길어졌을 때 요약해서 컨텍스트를 정리
- `/clear` - 대화 초기화

각 명령어의 자세한 사용법은 해당 파일을 참고하세요.
