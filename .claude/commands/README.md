# Commands

사용 가능한 Slash Commands 목록입니다.

## 초기 설정

- `/setup-workspace` - 워크스페이스 초기 설정 (CLAUDE.md 자동 생성)
- `/setup-google-calendar` - Google Calendar 연결 (선택, 자동 설치 + OAuth)

## Daily Workflow

- `/daily-note` - 오늘 Daily Note 생성/열기
- `/daily-review` - 하루 정리
- `/todo` - 할일 캡처/조회 (daily 기록에서 자동 추출도 겸함)

## AI 활용

- `/thinking-partner` - AI와 대화하며 생각 발전
- `/gather` - 정보 수집 모드
- `/reframe` - 이해 확인 모드
- `/truth` - 사실 기반 분석 모드

## 지식 관리 / 시스템

- `/idea [카테고리]` - 아이디어 저장
- `/create-command` - 커스텀 명령어 생성

---

각 명령어의 자세한 사용법은 해당 파일을 참고하세요.
