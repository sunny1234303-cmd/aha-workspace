## Claude Code란?

### 일반 Claude와의 차이

| 구분 | 일반 Claude (웹/앱) | Claude Code |
|------|---------------------|-------------|
| 위치 | 브라우저에서 실행 | 내 컴퓨터에서 실행 |
| 역할 | 대화만 가능 | 실제 작업 수행 |
| 결과 | "이렇게 하세요" 조언 | "다 했습니다" 실행 |

**일반 Claude**: "매출 데이터를 분석하려면 이렇게 하세요..."

**Claude Code**: "매출 데이터를 분석했습니다. 차트를 만들고 보고서를 저장했어요. 주요 인사이트는..."

> 컨설턴트가 조언만 하는 것 vs 직접 실행까지 해주는 것의 차이

## Part 1: 환경 준비

### 1-1. imi-workspace 클론

```bash
git clone https://github.com/Rhim80/imi-workspace.git
cd imi-workspace
```

### 1-2. Claude Code 실행

```bash
claude
```

처음 실행하면 로그인이 필요합니다:
```
/login
```

### 1-3. Python 설치 확인

데이터 분석을 위해 Python이 필요합니다:
```
"Python이 설치되어 있어? 없으면 설치해줘"
```

### 1-4. 현재 위치 확인

Claude에게 물어보세요:
```
"지금 어느 폴더에 있어?"
```

### 1-5. 경로 이해하기

컴퓨터는 폴더로 구성된 건물과 같습니다. Claude Code에게 어느 방에서 작업할지 알려줘야 합니다.

**경로 표현 방법**

| 표현 | Mac/Linux | Windows |
|------|-----------|---------|
| 홈 폴더 | `~` (예: `~/Downloads`) | `C:\Users\{사용자명}` |
| 현재 폴더 | `.` | `.` |
| 상위 폴더 | `..` | `..` |

**폴더 구조 예시**

**Mac/Linux:**
```
내 컴퓨터
└── Users
    └── 내이름 (📍 ~ 여기가 홈)
        ├── Downloads
        ├── Documents
        └── Desktop
```

**Windows:**
```
내 컴퓨터
└── C:\Users
    └── 내이름 (📍 홈 폴더)
        ├── Downloads
        ├── Documents
        └── Desktop
```

## Part 2: 폴더 정리 실습

### 목표
Downloads 폴더 (또는 지정된 폴더)를 Claude Code로 정리하기

### 왜 Downloads 폴더인가?
- 누구나 가지고 있고, 대부분 어지러움
- 실험해도 중요한 파일이 없음 (저위험)
- 결과가 바로 보임 (즉각적 만족감)

### 실습 순서

**Step 1**: 분석 요청

**Mac/Linux:**
```
"~/Downloads 폴더를 분석해서 어떤 파일들이 있는지 정리해줘"
```

**Windows:**
```
"C:\Users\{내이름}\Downloads 폴더를 분석해줘"
```
또는 간단하게: `"Downloads 폴더를 분석해줘"` (Claude가 경로 찾아줌)

**Step 2**: 정리 계획 요청
```
"이 파일들을 유형별로 정리하는 계획을 세워줘"
```

**Step 3**: 계획 확인 및 수정
```
"업무 관련 PDF는 별도 폴더로 분리해줘"
```

**Step 4**: 실행
```
"계획대로 정리해줘"
```

### 안전장치

**걱정하지 마세요:**
- Claude는 삭제 전 항상 확인을 요청합니다
- 파일을 "삭제"하지 않고 "이동"합니다
- "되돌려줘"로 언제든 복구 가능

**유용한 확인 명령:**
```
"방금 정리한 내용을 알려줘"
```
```
"세금 관련 파일 어디로 옮겼어?"
```
```
"마지막 작업 취소해줘"
```

## Part 3: 데이터 분석 실습 - 오피스원 주식회사

### 상황 설정

당신은 **오피스원 주식회사** 경영기획팀의 신입 사원입니다.

오피스원은 B2B 사무용품/사무환경 솔루션을 판매하는 회사로, 직원 약 50명, 연매출 약 50억원 규모입니다.

지금은 2025년 6월 30일, 2분기가 막 끝났습니다.

박 팀장이 다가와서 말합니다:

> "이번 주 금요일에 2분기 경영 회의야.
> CEO한테 보고할 자료 준비해야 하는데, 데이터 분석 좀 도와줄 수 있어?
> 영업팀, 마케팅팀, 인사팀에서 데이터 받아놨으니까 분석해서 인사이트 뽑아줘."

### 3-1. 샘플 데이터 위치로 이동

터미널에서:
```bash
cd 50-resources/sample-data
claude
```

### 3-2. 미션 1: 판매 실적 분석

**박 팀장:**
> "2분기 판매 실적을 분석해서 슬라이드 3장 분량으로 요약해줘.
> CEO가 좋아하는 건 숫자보다 인사이트야."

**Step 1**: 전체 현황 파악
```
"sample_sales_data.csv를 분석해서 2분기 판매 현황을 알려줘"
```

**Step 2**: 월별 추이 분석
```
"월별 매출 추이가 어때? 4월, 5월, 6월 비교해줘"
```

**Step 3**: 지역별 성과 비교
```
"지역별 매출을 비교해줘. 서울, 경기, 부산 중 어디가 가장 좋아?"
```

**Step 4**: 영업담당별 실적
```
"가장 실적이 좋은 영업담당은 누구야? 왜 그런 것 같아?"
```

**Step 5**: 인사이트 정리
```
"지금까지 분석한 내용을 CEO 보고용으로 3줄로 요약해줘"
```

### 3-3. 미션 2: 고객 현황 점검

**박 팀장:**
> "비활성 고객이 6곳이나 돼. CEO한테 이 얘기 나올 거야.
> 왜 떠났는지, 다시 데려올 수 있는지 분석해봐."

**Step 1**: VIP 고객 식별
```
"customer_data.csv에서 VIP 고객 (누적매출 상위 20%)을 찾아줘"
```

**Step 2**: 비활성 고객 분석
```
"비활성 고객은 몇 개월 전에 마지막 구매를 했어?"
```

**Step 3**: 이탈 패턴 파악
```
"이탈 위험이 있는 고객은 누구야? 공통점이 있어?"
```

**Step 4**: 재활성화 전략
```
"비활성 고객을 다시 데려오려면 어떻게 해야 할까?"
```

### 3-4. 미션 3: 마케팅 ROI 검토

**박 팀장:**
> "마케팅팀이 예산을 많이 썼는데 효과가 있었나 봐야 해.
> 채널별로 ROI 비교해서 다음 분기 예산 조정 근거를 만들어줘."

**Step 1**: 채널별 ROI 비교
```
"marketing_campaign_data.csv에서 ROI가 가장 높은 채널은?"
```

**Step 2**: 비효율 채널 식별
```
"예산 대비 효율이 낮은 채널은 어디야?"
```

**Step 3**: 예산 재배분 제안
```
"다음 분기에 예산을 늘려야 할 채널과 줄여야 할 채널을 추천해줘"
```

Claude Code가 잘 분석하려면 데이터가 다음 조건을 갖추면 좋습니다:

- 날짜 형식: YYYY-MM-DD (예: 2025-04-01)
- 명확한 컬럼 헤더
- 일관된 데이터 형식
- 병합된 셀 없음

## 유용한 명령어 모음

### 경로 관련
| 명령어 | Mac/Linux | Windows |
|--------|-----------|---------|
| Downloads 이동 | `cd ~/Downloads` | `cd C:\Users\{이름}\Downloads` |
| 상위 이동 | `cd ..` | `cd ..` |
| 현재 위치 | `pwd` | `cd` (경로만 표시) |

### Claude Code 내부
| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 |
| `/clear` | 대화 초기화 |
| `Ctrl+C` | 작업 중단 |
| `exit` | Claude Code 종료 |

## 트러블슈팅

### "command not found: claude"
- Claude Code가 설치되지 않았습니다
- 설치: `curl -fsSL claude.ai/install.sh | bash`

### 로그인이 안 돼요
- `/login` 입력 후 브라우저에서 로그인
- Claude Pro 구독 필요 ($20/월)

### 파일을 못 찾아요
- 현재 위치 확인: "지금 어디야?"
- 전체 경로 사용:
  - Mac/Linux: `~/Downloads/파일명.csv`
  - Windows: `C:\Users\{이름}\Downloads\파일명.csv`

### 분석이 안 돼요
- Python 설치 확인: "Python 설치되어 있어?"
- 파일 인코딩 확인: UTF-8 권장

## 흔한 오해

**"이건 개발자용 아닌가요?"**
→ 아닙니다. 파일, 데이터, 반복 작업이 있는 사람을 위한 도구입니다. 코드를 이해할 필요 없이 자연어로 대화하면 됩니다.

**"뭔가 잘못될까봐 무서워요"**
→ Claude Code는 변경 전 항상 허락을 구합니다. 모든 작업은 되돌릴 수 있습니다.

**"너무 기술적이에요"**
→ 이메일 쓸 수 있으면 Claude Code도 쓸 수 있습니다. 대화로 원하는 것을 설명하면 됩니다.

## 실제로 배운 것

이 실습을 마치면 여러분은 이미 다음을 익힌 것입니다:

- **파일 시스템 탐색**: 경로 개념, 폴더 구조 이해
- **패턴 인식**: 파일 유형별 분류
- **자동화 원리**: 반복 작업의 위임
- **데이터 분석**: 코드 없이 인사이트 도출

> 코딩을 배우는 게 아닙니다. AI에게 위임하는 법을 배우는 것입니다.
