## 1. RFM 분석

**출처**: Direct Marketing Association (1990년대 표준화)
**핵심**: 고객 가치를 3가지 행동 지표로 측정

### RFM 정의

| 지표 | 의미 | 측정 |
|------|------|------|
| **R**ecency | 최근성 | 마지막 구매 후 경과 일수 |
| **F**requency | 빈도 | 총 구매 횟수 |
| **M**onetary | 금액 | 총 구매 금액 |

### 점수 산정 방법

```
각 지표를 5분위로 나누어 1~5점 부여
- R: 최근일수록 높은 점수 (5점 = 가장 최근)
- F: 자주 살수록 높은 점수 (5점 = 가장 빈번)
- M: 많이 살수록 높은 점수 (5점 = 가장 고액)

총점 범위: 3점 ~ 15점
```

### 고객 세그먼트

| RFM 점수 | 세그먼트 | 특성 | 전략 |
|----------|----------|------|------|
| 12-15 | Champions | 최근 + 자주 + 고액 | 로열티 보상, 신제품 우선 제공 |
| 9-11 | Loyal | 자주 구매, 안정적 | 업셀/크로스셀 기회 |
| 6-8 | Potential | 가능성 있음 | 관계 강화, 프로모션 |
| 3-5 | At Risk | 이탈 위험 | 재활성화 캠페인 |

### 적용 질문

```
"customer_data.csv에서 RFM 분석을 해줘.
최근구매일로 R점수, 총주문수로 F점수, 누적매출로 M점수를 각각 5분위로 나눠서 계산해줘"

"RFM 총점 기준으로 Champions, Loyal, Potential, At Risk 고객을 분류해줘"
```

## 2. 고객 생애 가치 (CLV)

**출처**: Gupta & Lehmann (2003), Harvard Business Review
**핵심**: 고객 1명이 거래 기간 동안 창출하는 총 가치

### 간이 CLV 공식

```
CLV = (평균 구매 금액) × (구매 빈도/년) × (예상 거래 기간)

또는

CLV = (연간 매출) × (평균 거래 기간) × (이익률)
```

### B2B 간이 계산

```
CLV = 누적매출 / 거래기간(년) × 예상 잔여 거래기간

예시:
- 누적매출: 3,000만원
- 거래기간: 2년
- 연평균: 1,500만원
- 예상 잔여기간: 3년
- 예상 CLV: 1,500만원 × 3년 = 4,500만원
```

### 적용 질문

```
"고객별 연평균 매출을 계산하고, 예상 CLV를 추정해줘.
거래기간이 긴 고객은 잔여 기간을 3년, 신규 고객은 5년으로 가정해"

"CLV 상위 10개 고객 리스트를 만들어줘"
```

## 3. 고객 생명주기 (Customer Lifecycle)

**출처**: Kotler & Keller, Marketing Management
**핵심**: 고객과의 관계 단계별 맞춤 전략

### 생명주기 단계

```
Acquisition → Activation → Retention → Revenue → Referral
(획득)        (활성화)      (유지)      (매출화)   (추천)
    ↓             ↓           ↓           ↓         ↓
  신규고객     첫구매      재구매      VIP화     추천인
```

### 단계별 분류 기준 (B2B 예시)

| 단계 | 기준 | 액션 |
|------|------|------|
| 신규 | 첫구매 후 90일 이내 | 온보딩, 관계 구축 |
| 성장 | 구매 증가 추세 | 업셀/크로스셀 |
| 성숙 | 안정적 반복 구매 | 로열티 프로그램 |
| 위험 | 구매 감소/장기 미구매 | 재활성화 캠페인 |
| 이탈 | 6개월 이상 미구매 | Win-back 또는 정리 |

### 적용 질문

```
"고객을 생명주기 단계별로 분류해줘.
- 신규: 첫구매 후 90일 이내
- 성장: 최근 3개월 구매 증가
- 성숙: 6개월 이상 꾸준한 구매
- 위험: 최근 3개월 구매 감소
- 이탈: 6개월 이상 미구매"

"각 단계별 고객 수와 매출 비중을 알려줘"
```

## 4. 이탈 분석 (Churn Analysis)

### 이탈 신호 지표

| 신호 | 측정 방법 | 위험도 |
|------|-----------|--------|
| 구매 주기 2배 초과 | 평균 구매 주기 대비 | 높음 |
| 구매 금액 50% 감소 | 전기 대비 | 중간 |
| 문의/클레임 증가 | 최근 3개월 | 높음 |
| 결제 지연 | 평균 결제일 대비 | 중간 |

### 적용 질문

```
"비활성 고객들의 공통점을 분석해줘.
업종, 지역, 첫구매시점, 마지막 구매 전 패턴 등을 비교해줘"

"이탈 위험 고객을 조기 발견하려면 어떤 지표를 봐야 할까?"
```

## 5. 실습 적용 예시

### 오피스원 주식회사 데이터 분석

```
Step 1: RFM 분석
"customer_data.csv로 RFM 분석 해줘.
최근구매일, 총주문수, 누적매출 기준으로 점수 매기고 세그먼트 분류해줘"

Step 2: 고객 가치 분석
"누적매출 상위 20%가 전체 매출의 몇 %야? (파레토 확인)"

Step 3: 이탈 분석
"비활성 고객 10개사의 특징을 분석해줘.
공통점이 있어? 어떤 고객이 이탈 위험이 높을까?"

Step 4: 전략 제안
"이탈 고객을 다시 데려오려면 어떤 접근이 효과적일까?
세그먼트별로 다른 전략을 제안해줘"
```
