## 1. eNPS (Employee Net Promoter Score)

**출처**: Fred Reichheld (Bain & Company), 2003
**원래**: 고객 NPS를 직원 버전으로 적용
**핵심 질문**: "이 회사를 친구나 동료에게 일할 곳으로 추천하시겠습니까?"

### 점수 계산

```
0-6점: Detractors (비추천자)
7-8점: Passives (중립자)
9-10점: Promoters (추천자)

eNPS = 추천자 비율(%) - 비추천자 비율(%)
범위: -100 ~ +100
```

### 벤치마크

| eNPS | 해석 | 수준 |
|------|------|------|
| 50+ | 매우 우수 | 상위 10% |
| 30-50 | 우수 | 상위 30% |
| 10-30 | 양호 | 평균 |
| 0-10 | 개선 필요 | 평균 이하 |
| 0 미만 | 심각 | 즉시 조치 |

### 적용 질문

```
"employee_survey_data.csv에서 추천의향 점수로 eNPS를 계산해줘"
"부서별 eNPS를 비교해줘"
"비추천자(6점 이하)의 공통 특징이 뭐야?"
```

## 2. Gallup Q12 (참여도 12문항)

**출처**: Gallup Organization, 30년간 수백만 명 조사 기반
**핵심**: 12개 질문으로 직원 참여도(Engagement) 측정

### Q12 문항 영역

| 영역 | 문항 예시 | 측정 대상 |
|------|----------|----------|
| 기본 니즈 | 업무에 필요한 도구/자원 | 기초 환경 |
| 관리자 관계 | 관리자가 나를 인격체로 대함 | 관계/인정 |
| 소속감 | 내 의견이 중요하게 여겨짐 | 참여/기여 |
| 성장 | 성장/발전 기회가 있음 | 발전/학습 |

### 참여도 수준

```
Engaged (참여): 열정적, 생산적, 회사에 헌신
Not Engaged (비참여): 최소한의 노력, 기계적 업무
Actively Disengaged (적극적 불만): 불만 전파, 팀 사기 저하
```

### 벤치마크 (글로벌 평균)

| 수준 | 비율 | 특성 |
|------|------|------|
| Engaged | 20-30% | 핵심 인재 |
| Not Engaged | 50-60% | 개선 여지 |
| Actively Disengaged | 15-20% | 리스크 |

## 3. Herzberg 2요인 이론

**출처**: Frederick Herzberg (1959), "The Motivation to Work"
**핵심**: 만족 요인과 불만족 요인은 별개

### 두 가지 요인

```
위생요인 (Hygiene Factors) - 불만족 방지
┌─────────────────────────────────────┐
│ 급여, 복리후생, 근무환경, 회사정책, │
│ 관리감독, 대인관계, 고용안정       │
└─────────────────────────────────────┘
→ 부족하면 불만족, 충족해도 만족은 아님

동기요인 (Motivators) - 만족/동기부여
┌─────────────────────────────────────┐
│ 성취감, 인정, 업무 자체, 책임감,   │
│ 성장기회, 승진                     │
└─────────────────────────────────────┘
→ 충족되면 만족, 부족해도 불만족은 아님
```

### 분석 적용

```
설문 항목 분류:
- 위생요인: 워라밸, 보상만족도, 근무환경
- 동기요인: 성장기회, 관리자역량, 교육만족도

우선순위:
1. 위생요인 불만족 해소 (기본)
2. 동기요인 강화 (성장)
```

## 4. 만족도 분석 매트릭스

### 중요도-만족도 매트릭스 (IPA)

**출처**: Martilla & James (1977)

```
                     만족도
                 높음          낮음
         ┌───────────────┬───────────────┐
    높   │   유지 강화    │  집중 개선    │
 중 음   │ (Keep up)     │ (Concentrate) │
 요      ├───────────────┼───────────────┤
 도 낮   │  과잉 투자    │   저우선순위   │
    음   │ (Overkill)    │ (Low Priority)│
         └───────────────┴───────────────┘
```

### 적용

```
1. 각 항목의 평균 만족도 계산
2. 중요도는 추천의향과의 상관계수로 추정
3. 4분면 배치하여 우선순위 결정
```

## 5. 부서/그룹별 비교 분석

### 비교 차원

| 차원 | 비교 기준 | 인사이트 |
|------|----------|----------|
| 부서별 | 개발팀 vs 영업팀 | 직무 특성 영향 |
| 직급별 | 주니어 vs 시니어 | 경력단계 니즈 |
| 근속별 | 신입 vs 장기근속 | 조직 적응도 |
| 근무형태별 | 재택 vs 출근 | 환경 선호도 |

### 적용 질문

```
"부서별 평균 만족도를 모든 항목에 대해 비교해줘"
"직급별로 가장 차이가 큰 항목은 뭐야?"
"재택 선호자와 출근 선호자의 만족도 차이가 있어?"
```

## 6. 이직 위험 예측

### 이직 신호 지표

| 신호 | 측정 | 위험도 |
|------|------|--------|
| 전체 만족도 6점 이하 | 설문 | 높음 |
| 추천의향 6점 이하 | 설문 | 높음 |
| 성장기회 5점 이하 | 설문 | 중간 |
| 최근 성과 하락 | 인사 데이터 | 중간 |

### 적용 질문

```
"전체만족도 6점 이하인 직원의 특징을 분석해줘"
"이직 위험이 높아 보이는 그룹은 어디야?"
"성장기회 점수가 낮은 부서는 어디야?"
```

## 7. 실습 적용 예시

### 오피스원 주식회사 데이터 분석

```
Step 1: eNPS 계산
"employee_survey_data.csv에서 추천의향 점수로 eNPS를 계산해줘.
전체, 부서별, 직급별로 나눠서"

Step 2: 만족도 항목 비교
"모든 만족도 항목의 평균을 계산하고, 가장 높은/낮은 항목을 알려줘"

Step 3: 부서별 비교
"부서별 만족도를 비교해줘. 어떤 부서가 가장 높고 낮아?"

Step 4: 개선점 도출
"가장 개선이 필요한 영역 3가지를 추천해줘.
데이터 근거와 함께 설명해줘"

Step 5: 근무형태 분석
"재택 선호자와 출근 선호자의 만족도 차이가 있어?
어떤 항목에서 가장 차이가 나?"
```
