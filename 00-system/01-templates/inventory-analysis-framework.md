## 1. ABC 분석 (파레토 기반 재고 분류)

**출처**: General Electric (1950년대), 재고관리 표준 기법
**핵심**: 중요도에 따라 재고를 차등 관리

### ABC 분류 기준

| 등급 | 품목 비율 | 매출/비용 비중 | 관리 수준 |
|------|-----------|----------------|-----------|
| A | 10-20% | 70-80% | 엄격 관리 |
| B | 20-30% | 15-20% | 중간 관리 |
| C | 50-70% | 5-10% | 느슨 관리 |

### 분류 기준 선택

```
1. 매출액 기준: 판매 금액이 큰 품목
2. 마진 기준: 수익 기여도가 큰 품목
3. 이동량 기준: 판매 빈도가 높은 품목
4. 전략적 기준: 핵심 제품, 고객 요구
```

### 적용 질문

```

"A등급 품목은 몇 개야? 전체 매출의 몇 %를 차지해?"
"C등급인데 재고가 과다한 품목은 뭐가 있어?"
```

## 2. 재고 회전율 & 재고일수

**출처**: 재무회계/운영관리 표준 지표

### 핵심 공식

```
재고 회전율 = 매출원가 / 평균 재고
           = 판매량 / 평균 재고 수량

재고일수 (DOH) = 365 / 재고 회전율
              = (현재고 / 일평균 판매량)
```

### 해석 기준

| 재고일수 | 상태 | 액션 |
|----------|------|------|
| 7일 미만 | 재고 부족 위험 | 긴급 발주 |
| 7-14일 | 적정 범위 | 모니터링 |
| 14-30일 | 약간 과다 | 판촉 검토 |
| 30일 초과 | 과다 재고 | 할인/처분 검토 |

### 적용 질문

```
"각 품목의 재고일수를 계산해줘. 30일 판매량 기준으로"
"재고일수가 7일 미만인 품목은 뭐가 있어?"
"재고일수 30일 초과 품목의 재고 금액 합계는?"
```

## 3. 재주문점 (ROP) & 안전재고

**출처**: 운영관리 표준 모델

### 재주문점 공식

```
ROP = (일평균 수요 × 리드타임) + 안전재고

예시:
- 일평균 판매: 10개
- 리드타임: 7일
- 안전재고: 20개
- ROP = (10 × 7) + 20 = 90개
→ 재고가 90개 이하가 되면 발주
```

### 안전재고 산정

```
안전재고 = Z × σ × √리드타임

Z: 서비스 수준에 따른 계수
   95% 서비스 → Z = 1.65
   99% 서비스 → Z = 2.33
σ: 수요 표준편차
```

### 간이 안전재고 (실무용)

```
안전재고 = 일평균 수요 × 리드타임 × 안전계수

안전계수:
- A등급 품목: 0.5 (높은 서비스 수준)
- B등급 품목: 0.3
- C등급 품목: 0.2
```

## 4. 재고 상태 분류 매트릭스

### 긴급도 판단

```
                    판매 속도
                 빠름          느림
         ┌───────────────┬───────────────┐
    낮   │    긴급        │   주의         │
 재 음   │ (즉시 발주)    │ (계획 발주)    │
 고      ├───────────────┼───────────────┤
    높   │    정상        │   과다         │
    음   │ (모니터링)     │ (판촉 필요)    │
         └───────────────┴───────────────┘
```

### 상태별 정의

| 상태 | 조건 | 우선순위 |
|------|------|----------|
| 품절 | 현재고 = 0 | 최우선 |
| 긴급 | 현재고 < 재주문점 × 0.5 | 높음 |
| 재고부족 | 재주문점 ≤ 현재고 < 재주문기준 | 중간 |
| 정상 | 재주문기준 ≤ 현재고 ≤ 최대재고 | 낮음 |
| 과다 | 현재고 > 최대재고 | 주의 |

## 5. 발주 우선순위 프레임워크

### 우선순위 점수 산정

```
우선순위 점수 = (긴급도 × 3) + (매출 기여도 × 2) + (마진율 × 1)

긴급도: 품절=5, 긴급=4, 재고부족=3, 정상=1
매출기여도: A등급=5, B등급=3, C등급=1
마진율: 고마진=5, 중마진=3, 저마진=1
```

### 적용 질문

```
"재고 상태가 '품절' 또는 '긴급'인 품목을 찾아줘"
"발주 우선순위를 정해줘. 긴급도와 매출 기여도를 고려해서"
"긴급 발주 품목의 예상 발주 금액은 얼마야?"
```

## 6. 공급업체 리스크 분석

### 의존도 분석

```
공급업체 의존도 = 해당 업체 품목 수 / 전체 품목 수
               = 해당 업체 매출 비중

위험 수준:
- 30% 이상: 높은 의존도, 리스크 관리 필요
- 15-30%: 중간 의존도
- 15% 미만: 분산된 구조
```

### 적용 질문

```
"공급업체별 품목 수와 매출 비중을 알려줘"
"가장 의존도가 높은 공급업체는 어디야?"
"긴급 품목의 공급업체가 겹치는 게 있어? 리스크가 있을까?"
```
