## 1. Pareto 분석 (80/20 법칙)

**출처**: 이탈리아 경제학자 빌프레도 파레토 (1896)
**핵심**: 전체 결과의 80%는 20%의 원인에서 발생

### 적용 방법

```
1. 매출 기여도 분석
   - 상위 20% 제품/고객이 전체 매출의 몇 %를 차지하는가?
   - 집중 관리 대상 식별

2. 분석 질문
   "제품별 매출을 내림차순 정렬하고, 상위 20%가 전체 매출의 몇 %인지 계산해줘"
   "고객사별 누적 매출 상위 20%를 찾아줘"
```

### 결과 해석

| 비율 | 의미 | 액션 |
|------|------|------|
| 상위 20%가 80% 이상 | 전형적 파레토 | 상위 집중 관리 |
| 상위 20%가 60% 미만 | 분산된 구조 | 포트폴리오 다각화 유지 |
| 상위 5%가 50% 이상 | 과도한 의존 | 리스크 분산 필요 |

## 2. 성장률-점유율 매트릭스 (BCG Matrix 응용)

**출처**: Boston Consulting Group (1970)
**원래 용도**: 사업 포트폴리오 분석
**응용**: 제품/지역/담당자 포트폴리오 분석

### 4분면 분류

```
                    성장률 (전월/전분기 대비)
                    높음              낮음
         ┌─────────────────┬─────────────────┐
    높   │      Star       │   Cash Cow      │
 매 음   │  (성장 + 고매출) │ (안정 + 고매출)  │
 출      ├─────────────────┼─────────────────┤
 비 낮   │  Question Mark  │      Dog        │
 중 음   │  (성장 + 저매출) │ (정체 + 저매출)  │
         └─────────────────┴─────────────────┘
```

### 적용 질문

```
"지역별로 매출 비중과 전월 대비 성장률을 계산해서 4분면으로 분류해줘"
"영업담당별 실적을 Star/Cash Cow/Question Mark/Dog로 나눠줘"
```

### 전략 제안

| 분류 | 특성 | 전략 |
|------|------|------|
| Star | 고성장 + 고매출 | 투자 확대, 리소스 집중 |
| Cash Cow | 저성장 + 고매출 | 효율화, 수익 극대화 |
| Question Mark | 고성장 + 저매출 | 선택적 투자, 모니터링 |
| Dog | 저성장 + 저매출 | 철수 검토, 최소 투자 |

## 3. 차이 분석 (Variance Analysis)

**출처**: 관리회계 표준 기법
**용도**: 계획 vs 실적, 전기 vs 당기 비교

### 분석 프레임

```
차이 = 실적 - 기준
     = (가격 차이) + (수량 차이) + (믹스 차이)

1. 가격 차이: 단가 변동 영향
2. 수량 차이: 판매량 변동 영향
3. 믹스 차이: 제품 구성 변동 영향
```

### 적용 질문

```
"4월 대비 6월 매출 차이를 분석해줘. 수량 변화인지 단가 변화인지 구분해줘"
"전분기 대비 매출 증감의 원인을 제품별로 분해해줘"
```

### 결과 정리 템플릿

```markdown
## 매출 차이 분석 (기준: OO vs 비교: OO)

총 차이: +/- OOO원 (OO%)

### 원인 분해
1. 수량 효과: +/- OOO원
   - 주요 원인: [구체적 설명]
2. 단가 효과: +/- OOO원
   - 주요 원인: [구체적 설명]
3. 믹스 효과: +/- OOO원
   - 주요 원인: [구체적 설명]

### 시사점
- [인사이트 1]
- [인사이트 2]
```

## 4. 실습 적용 예시

### 오피스원 주식회사 데이터 분석

```

"5월 매출이 4월보다 낮은데, 수량 감소인지 단가 하락인지 분석해줘"
```
