# 마케팅 ROI 분석 프레임워크

> 기반: ROAS, CAC/LTV, Marketing Mix Modeling, Attribution

## 1. 핵심 마케팅 지표

### ROI vs ROAS

| 지표 | 공식 | 용도 |
|------|------|------|
| **ROI** | (수익 - 비용) / 비용 × 100% | 순이익 기준 효율성 |
| **ROAS** | 매출 / 광고비 | 매출 기준 효율성 |

```
예시:
광고비 100만원, 매출 500만원, 원가 300만원

ROAS = 500만 / 100만 = 5.0 (500%)
ROI = (500만 - 300만 - 100만) / 100만 = 100%
```

### 고객 획득 비용 (CAC)

**출처**: SaaS/스타트업 표준 지표, David Skok (Matrix Partners)

```
CAC = 총 마케팅/영업 비용 / 신규 고객 수

채널별 CAC = 채널별 비용 / 채널별 획득 고객 수
```

### CAC:LTV 비율

| 비율 | 해석 | 액션 |
|------|------|------|
| 1:1 미만 | 손실 구조 | 즉시 개선 필요 |
| 1:1 ~ 1:3 | 수익성 낮음 | 효율화 필요 |
| 1:3 이상 | 건전한 구조 | 투자 확대 가능 |
| 1:5 이상 | 매우 효율적 | 성장에 재투자 |

## 2. 채널 효율성 분석

### Marketing Mix 평가 매트릭스

```
                      전환율
                 높음           낮음
         ┌───────────────┬───────────────┐
    높   │   효율 채널    │  인지도 채널   │
 비 음   │ (핵심 투자)    │ (브랜딩 목적)  │
 용      ├───────────────┼───────────────┤
    낮   │   테스트 채널  │   비효율 채널  │
    음   │ (확대 검토)    │ (축소/폐지)    │
         └───────────────┴───────────────┘
```

### 채널별 벤치마크 (B2B 기준)

| 채널 | 평균 ROAS | 특성 |
|------|-----------|------|
| 검색 광고 (SEM) | 4-8x | 의도 기반, 높은 전환 |
| 이메일 마케팅 | 30-40x | 기존 고객, 저비용 |
| 콘텐츠 마케팅 | 장기 | SEO, 신뢰 구축 |
| 전시회/이벤트 | 1-3x | 네트워킹, 리드 발굴 |
| SNS 광고 | 2-5x | 인지도, 리타겟팅 |

### 적용 질문

```
"marketing_campaign_data.csv에서 채널별 ROAS를 계산해줘"
"CAC가 가장 낮은 채널은 어디야?"
"전환율과 비용을 기준으로 채널을 4분면으로 분류해줘"
```

## 3. Attribution 모델 (기여도 배분)

**출처**: Google Analytics, 디지털 마케팅 표준

### 주요 모델

| 모델 | 설명 | 적합 상황 |
|------|------|----------|
| Last Touch | 마지막 접점에 100% | 단순, 즉각 전환 |
| First Touch | 첫 접점에 100% | 인지도 측정 |
| Linear | 모든 접점 균등 | 다채널 균형 |
| Time Decay | 최근 접점에 가중치 | B2B 긴 구매 주기 |
| Position Based | 첫/끝 40%, 중간 20% | 복합 퍼널 |

### B2B 권장 모델

```
Position Based (U-shaped) 권장 이유:
1. 첫 접점: 리드 발굴의 공로
2. 마지막 접점: 전환 결정의 공로
3. 중간 접점: 육성(Nurturing) 역할
```

## 4. 예산 재배분 프레임워크

### 효율성 기반 배분

```
1. 현재 채널별 ROAS 계산
2. ROAS 순위 매기기
3. 재배분 원칙 적용:
   - ROAS 상위 채널: 예산 20-30% 증액
   - ROAS 중위 채널: 현상 유지 또는 최적화
   - ROAS 하위 채널: 예산 30-50% 감축 또는 폐지
```

### 포트폴리오 관점

```
권장 예산 배분:
- 검증된 채널 (Proven): 60-70%
- 성장 채널 (Growing): 20-30%
- 실험 채널 (Testing): 5-10%
```

### 적용 질문

```
"채널별 ROI 순위를 매기고, 예산 재배분안을 제안해줘"
"비효율 채널의 예산을 효율 채널로 옮기면 예상 매출 증가는 얼마야?"
"다음 분기 마케팅 예산 배분을 추천해줘"
```

## 6. 실습 적용 예시

### 오피스원 주식회사 데이터 분석

```
Step 1: 채널별 ROI 계산
"marketing_campaign_data.csv에서 채널별 총 비용, 총 매출, ROI를 계산해줘"

Step 2: 효율성 순위
"ROI가 가장 높은 채널 3개와 가장 낮은 채널 3개를 알려줘"

Step 3: 벤치마크 비교
"우리 채널별 ROAS가 업계 평균과 비교해서 어때?"

Step 4: 예산 재배분
"현재 예산 배분을 ROI 기준으로 최적화하면 어떻게 바뀌어야 해?
구체적인 금액으로 제안해줘"
```
