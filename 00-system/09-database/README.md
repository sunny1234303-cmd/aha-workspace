# Database

표 형태로 관리할 목록(고객, 연락처, 프로젝트 트래커, 재고 등)을 마크다운 파일로 두는 자리입니다.

[[setup-workspace|/setup-workspace]]를 실행하면 여기에 첫 데이터베이스 파일이 자동으로 만들어집니다. 별도 연동 없이 마크다운 표만으로 바로 씁니다.

## 예시

```markdown
# 고객 목록

| 이름 | 연락처 | 상태 | 메모 |
|---|---|---|---|
|   |   |   |   |
```

## 구글시트로 옮기고 싶다면

지금은 자동 연동을 기본 제공하지 않습니다. Claude에게 "이 표를 구글시트랑 연동하고 싶어"라고 요청하면 연동 방법을 같이 만들어줍니다.
