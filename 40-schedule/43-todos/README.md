# 43 Todos

할 일 정리 폴더입니다.

## 파일

- `active-todos.md` — 진행 중인 할 일 (Inbox / Today / Overdue로 구분)
- `completed-todos.md` — 완료 아카이브

## 사용법

`/오늘` 명령어 하나로 추가·조회·정리를 다 합니다. `41-daily/`에 자유롭게 쓴 기록에서 Claude가 할일을 찾아 여기로 옮겨줍니다.

## Todo 관리 원칙

1. 작은 단위로 나누기
2. 우선순위 설정
3. 완료 시 체크 ✅
4. 매주 정리 (`/오늘 overdue`로 방치된 것 확인)

자세한 사용법은 `00-system/03-guides/TODO-SYSTEM-GUIDE.md` 참고.
