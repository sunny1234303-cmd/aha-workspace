# 40 Schedule

일정관리 공간입니다 — 매일 기록하고, 할일을 챙기고, 주간으로 돌아보는 곳.

## 폴더 구조

- `41-daily/` - 일별 기록 (자유롭게 쓰되, 할일도 같이 섞어서 적으면 됨)
- `42-weekly/` - 주간 리뷰
- `43-todos/` - 할일 정리 (직접 나누지 않아도, `/todo`가 daily 기록에서 자동으로 추출·정리)

## 쓰는 법

1. 매일 `/daily-note`로 오늘 기록을 시작한다 — "오늘 할 일: ~" 처럼 자연스럽게 섞어 써도 됨
2. `/todo`를 실행하면 daily 기록에서 할일로 보이는 문장을 찾아 `43-todos/active-todos.md`에 정리해줌
3. 주 1회 `42-weekly/`에 주간 리뷰 작성

## 특징

- **자유로운 형식**: 정해진 형식 없이 자유롭게 작성
- **자동 정리**: 할일 분류는 직접 안 해도 됨 — Claude가 대신 정리
- **시간 기반**: 날짜별로 관리되는 내용
